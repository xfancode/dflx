# Deflex Studio v4.1

**Deflex Studio** — это лёгкая, мощная и портативная среда разработки, которая работает прямо в браузере.

---

## Возможности

- 🖥️ **EXE (HTML)** — конвертируй HTML в EXE,
- 🪟 **Windows Forms** — визуальный редактор с drag&drop,
- 📦 **BAT → EXE** — превращай батники в исполняемые файлы,
- 📜 **HTML → JS** — компиляция HTML в JavaScript,
- 🐍 **Python** — пиши Python-код прямо в браузере,
- #️⃣ **C# → EXE** — компиляция C# кода в EXE.

---

## Как использовать

1. Открой [Deflex Studio](https://xfancode.github.io/dflx)
2. Выбери платформу (C#, Python, WinForms и т.д.)
3. Напиши код или нарисуй интерфейс
4. Нажми **"Компилировать"** или **"Скачать"**
5. Готово! Исполняемый файл у тебя на рабочем столе

---

## Технологии

- **Frontend**: HTML + CSS + JavaScript (без фреймворков)
- **Хранение**: localStorage (проекты сохраняются локально, не отправляются на сервер)
- **Импорт/Экспорт**: .dflx

---

## Формат проектов (.dflx)

Проекты экспортируются в `.dflx` — это ZIP-архив с:
- `manifest.json` — информация о проекте
- `LICENSE.TXT` — лицензия
- `source/code.txt` — код (для HTML/BAT/Python/C#)
- `winforms/controls.json` — визуальные компоненты (для WinForms)

---

## Безопасность
